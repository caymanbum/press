# DNA - understanding the numbers

DNA is made up of 4 nitrogenous bases

 - Adenine  (A)
 - Thymine  (T)
 - Cytosine (C)
 - Guanine  (G)

They combine in a very particular way.

 - A always pairs with T
 - C always pairs with G

Effectively they are a code - base 4 which can be expressed as data.

The human genome consists of 3.2 billion base pairs

The COVID virus was slightly different - it's a code expressed in [RNA](rna.md) and it
only had a mere 30,000 base pairs.  Most of the viruses which affect Humans are [single
stranded RNA viruses](virus.md).
