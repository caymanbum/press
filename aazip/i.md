# Protect Our Legacy

Iguana 6 is a valuable legacy digital asset that deserves protection. Safeguarding our legacy systems ensures we continue to benefit from their reliability while modernizing their capabilities.

To do this, we need to:

- **Adopt a new, superior installation format**—Transforming Iguana Classic into a single binary, similar to Iguana X.  
- **Implement stricter control over configuration files**.  
- **Upgrade logging and queuing formats** for greater efficiency and transparency.  
- **Provide a secure location for storing authentication information**.

## Enabling Virtualization

To protect legacy systems, we use virtualization. By creating a [custom environment](fantasy.md), we can “trick” legacy applications into believing they're operating within a real file system, while actually protecting their critical components.

## Enter Cosmopolitan: A Revolutionary Technology

We’re embracing an innovative tool called [Cosmopolitan](cosmopolitan/i.md), invented by a brilliant and unconventional mind. Cosmopolitan allows us to compile binaries that run on all major operating systems, dramatically reducing friction in deployment and updates.

## Securing Plain Text LLP HL7 Data Feeds

Many healthcare applications only support unencrypted HL7 data feeds—a risky assumption in today’s security landscape. We can’t count on attackers being kept out by the firewall anymore. Our new approach allows even legacy HL7 data feeds to be secured, bringing them up to modern standards.
